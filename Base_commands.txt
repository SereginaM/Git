Базовые команды:

pwd			где я сейчас
clear			очисть историю
date			вывод даты
finde -name 		поиск файла по имени

cd имя_папки 		перейти в папку
cd ..			перейти на уровень выше

mkdir имя_папки			создать новую папку
mkdir имя_папки1...имя_папкиN  	создать больше одной папки

touch имя_файла.расширение 				создать пустой файл
touch имя_файла1.расширение...имя_файлаN.расширение 	создать больше одного пустого файла

ls 		отобразить содержимое папки
ls -la 		отобразить все содержимое папки, включая скрытые файлы

mv имя_файла.расширение путь для перемещения	переместить файл



Работа с файлами:

vim название_файла 			открыть редактирование файла

Выход из файла нажать кнопку "ESC" и ввести команду:
:x - выйти из файла с сохранением
:w - сохранит изменения без выхода
:q - выйти из файла без сохранения

Начать редактировать файл A/insert
